## 8. Combined Example

Here's a real-world example combining multiple components:

function Hello({ name }) {
  return (
    <div className="hello">

    </div>

}

export default Hello;`}

## Summary

Available MDX components:

| Component | Usage | Props |
|-----------|-------|-------|
| `Callout` | `<Callout type="info">` | `type`: info/warning/error/success/tip, `title` (optional) |
| `CodeBlock` | `<CodeBlock filename="...">` | `filename`, `highlightLines` |
| `YouTube` | `<YouTube videoId="...">` | `videoId`, `title`, `start` |
| `Mermaid` | `<Mermaid chart={...}>` | `chart` (Mermaid syntax string) |
| `Image` | `<Image src="..." />` | `src`, `alt`, `width`, `height`, `priority` |

**Note:** All these features work in both light and dark mode! üåì
